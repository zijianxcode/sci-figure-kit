# Data Model: Paper Index Landing Page

## `meta.json` — Per-Paper Metadata

Each paper directory under `papers/` contains a `meta.json` file:

```json
{
  "title": "Attention Is All You Need",
  "authors": [
    "Ashish Vaswani",
    "Noam Shazeer",
    "Niki Parmar",
    "Jakob Uszkoreit",
    "Llion Jones",
    "Aidan N. Gomez",
    "Lukasz Kaiser",
    "Illia Polosukhin"
  ],
  "year": 2017,
  "venue": "NeurIPS 2017",
  "description": "Introduces the Transformer architecture, replacing recurrence with self-attention for sequence modeling.",
  "thumbnail": "page-1.png",
  "detail_page": "final.html"
}
```

### Field Specification

| Field | Type | Required | Notes |
|-------|------|----------|-------|
| `title` | string | Yes | Full paper title |
| `authors` | string[] | Yes | Ordered list of author names |
| `year` | integer | Yes | Publication year |
| `venue` | string | No | Journal/conference name. Omitted from card if absent. |
| `description` | string | No | One-sentence summary. Omitted from card if absent. |
| `thumbnail` | string | No | Path relative to paper directory. Falls back to generated placeholder if absent or file missing. |
| `detail_page` | string | Yes | Path relative to paper directory, e.g., `final.html` |

## Manifest — Discovered Papers Index

Generated by `scripts/discover-papers.sh`, output as `papers/manifest.json`:

```json
{
  "papers": [
    {
      "id": "attention-is-all-you-need",
      "title": "Attention Is All You Need",
      "authors": ["Ashish Vaswani", "Noam Shazeer", "..."],
      "year": 2017,
      "venue": "NeurIPS 2017",
      "description": "Introduces the Transformer architecture...",
      "thumbnail": "attention-is-all-you-need/page-1.png",
      "detail_url": "attention-is-all-you-need/final.html"
    }
  ]
}
```

- `id`: Directory name (URL-safe slug)
- All paths relative to `papers/` directory
- Papers sorted by year descending, then by title alphabetically
