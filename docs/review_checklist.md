# 出图前检查清单

## 内容一致性

- 图的范围是否与正文图题一致？
- 图中是否新增了正文没有支撑的概念？
- 图中是否遗漏了正文强调的关键区分？
- 图 2.3 类编码图是否正确区分“状态”和“指标”？

## 简洁性

- 是否存在整句解释被塞进图内？
- 是否像正文段落的盒子化版本？
- 是否可以删掉图例或底部说明？
- 是否有重复表达：正文已经说了，图里又说一遍？

## 可读性

- A4 宽度下文字是否清楚？
- 图题和图片是否在同一页？
- 是否存在大面积空白？
- 图片是否是紧凑矩形？

## 视觉一致性

- 同一层级是否使用同一形状？
- 颜色是否用于分组，而不是装饰？
- 主流程、依赖、反馈、动态切换是否用不同线型？
- 是否避免了过度饱和色和厚重黑色边框？

## 导出检查

```bash
python3 build_figures.py
```

检查输出目录：

```text
outputs/current/
```

若要嵌入论文 PDF，再检查：

- PNG 分辨率是否足够。
- PDF 中图片是否被压得过小。
- 图题是否和图像分页分离。
- 是否同步保留 `.svg`，并放入 `figma_editable/`。
- Figma 中导入 SVG 后，文字和形状是否可以拆开编辑。
